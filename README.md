## ✨ Destaques da página

* **🧭 Navbar fixa:** navegação por âncoras com scroll suave entre as seções.
* **🌗 Dark/Light Mode:** tema dinâmico com persistência da preferência (LocalStorage).
* **⌨️ Efeito Typewriter:** digitação que cicla entre as áreas de atuação (JS puro).
* **🎨 UI Tech:** fundo em grid "papel milimetrado" e cards com Glassmorphism.
* **🎠 Carrossel de Projetos:** contagem de cards dinâmica e responsiva.
* **📱 Responsividade:** layout adaptável construído com Tailwind CSS.
